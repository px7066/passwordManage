# 服务器密码管理工具 - 规格说明书

## 1. 项目概述

- **项目名称**: Server Password Manager
- **类型**: Electron桌面应用
- **核心功能**: 管理服务器登录凭证，支持密码生成、导入导出、SSH终端连接
- **目标用户**: 运维工程师、系统管理员

## 2. 数据模型

### ServerEntry (服务器条目)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | UUID，唯一标识 |
| type | string | 服务器类型 (如: MySQL, Redis, Nginx, SSH等) |
| isCluster | boolean | 是否集群 |
| ips | string[] | IP数组 (集群时) 或单个IP |
| port | number | 端口号 |
| username | string | 用户名 |
| password | string | 密码 |
| remark | string | 备注 |
| autoCommands | string | 连接后自动执行的命令 (每行一条) |
| createdAt | string | 创建时间 |
| updatedAt | string | 更新时间 |

## 4. 功能规格

### 4.1 新增服务器

- 点击"新增"按钮打开编辑表单
- 必填: 类型、IP、端口、用户名、密码
- 类型下拉选项: SSH, MySQL, Redis, PostgreSQL, MongoDB, Nginx, Apache, Other
- 集群开关切换影响IP输入方式:
  - 关闭: 单个IP输入框
  - 开启: IP数组输入(每行一个)

### 4.2 密码生成规则

- 字符集:
  - 数字: 0-9 (10个)
  - 大写字母: A-Z (26个)
  - 小写字母: a-z (26个)
  - 特殊字符: !@#$%^&*()_+-=[]{}|;:,.<>? (约24个)
- 比例: 特殊字符:其他字符 = 1:7
- 示例(16位密码): 约2个特殊字符 + 14个其他字符
- 保证至少包含1个大写、1个小写、1个数字、1个特殊字符

### 4.3 导出功能

- 支持格式: CSV, Excel (.xlsx)
- 导出所有记录
- 文件名格式: servers_export_YYYYMMDD_HHmmss.{csv|xlsx}

### 4.4 导入功能

- 支持格式: CSV, Excel (.xlsx)
- 导入前显示预览
- 字段映射校验

### 4.5 删除功能

- 列表项删除按钮 (悬停显示)
- 删除前确认对话框

### 4.6 SSH终端功能

- 点击连接按钮建立SSH连接
- 多Tab终端界面
- 支持本地终端
- 自动执行命令功能

### 4.7 数据存储

- 本地JSON文件存储
- 文件路径: 用户数据目录/server_passwords.json
- 自动保存
