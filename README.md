# 服务器密码管理工具

一个基于 Electron 的桌面应用，用于安全管理服务器登录凭证，支持密码生成、SSH远程连接、SFTP文件传输等功能。

## 功能特性

### 服务器管理
- 支持多种服务器类型：SSH、MySQL、Redis、PostgreSQL、MongoDB、Nginx、Apache、网页等
- 集群模式支持（多IP管理）
- 服务器锁定功能（右键锁定后可防止误删）
- 密码生成器（可配置长度和字符类型）
- 服务器导入导出（支持 CSV 和 Excel 格式）

### SSH 终端
- 多 Tab 终端界面
- 支持本地终端和 SSH 远程连接
- 自动执行连接后的命令
- 终端主题：xterm dark

### SFTP 文件传输
- 本地目录浏览和远程目录浏览
- 文件上传下载
- 支持右键删除本地或远程文件
- 上传目录：使用当前本地目录
- 下载目录：使用当前本地目录

### 数据安全
- 本地 JSON 文件加密存储
- 锁定功能防止配置被修改或删除
- 密码强度检测

## 安装运行

### 环境要求
- Node.js 16+
- npm 或 yarn

### 安装步骤

```bash
# 克隆项目
git clone <项目地址>
cd passwordManage

# 安装依赖
npm install

# 运行应用
npm start
```

### 构建打包

```bash
# 构建 Windows 可执行文件
npm run build:win
```

构建完成的可执行文件在 `dist/` 目录。

## 使用说明

### 添加服务器
1. 点击左上角「新增」按钮
2. 选择服务器类型
3. 填写 IP 地址、端口、用户名
4. 点击「生成」按钮创建密码，或手动输入
5. 点击「保存」

### 连接服务器
1. 在服务器列表中点击连接按钮（绿色链接图标）
2. SSH 类型会打开终端并建立连接
3. 网页类型会用默认浏览器打开网址

### 文件传输
1. 点击服务器右侧的 SFTP 图标打开传输面板
2. 左侧为本地文件目录，右侧为远程文件目录
3. 选择本地文件和远程目录后，点击「上传」或「下载」
4. 右键点击文件可删除本地或远程文件

### 锁定服务器
1. 右键点击服务器
2. 选择「锁定」
3. 锁定后的服务器：
   - 连接和 SFTP 功能正常
   - 无法编辑配置
   - 无法删除（需先解锁）

### 导入导出
- **导出**：点击「导出」按钮，选择格式（CSV/Excel）
- **导入**：点击「导入」按钮，选择文件，预览后确认

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新增服务器 |
| `Ctrl+I` | 导入 |
| `Ctrl+E` | 导出 |
| `Delete` | 删除选中服务器 |

## 数据存储

数据存储在用户数据目录：
- Windows: `%APPDATA%\passwordManage\server_passwords.json`
- macOS: `~/Library/Application Support/passwordManage/server_passwords.json`
- Linux: `~/.config/passwordManage/server_passwords.json`

## 技术栈

- **框架**: Electron
- **前端**: HTML5 + CSS3 + Vanilla JavaScript
- **终端**: xterm.js
- **SSH**: ssh2
- **打包**: electron-builder
